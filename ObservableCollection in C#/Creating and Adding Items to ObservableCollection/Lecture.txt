Creating and Adding Items to ObservableCollection<T>
Objective
Learn to create an ObservableCollection<T> and add items to it.

Full Program Example
using System;
using System.Collections.ObjectModel;


class Program
{
    static void Main()
    {
        // Creating an ObservableCollection of strings
        ObservableCollection<string> names = new ObservableCollection<string>();

        // Adding items to the collection
        names.Add("Alice");
        names.Add("Bob");
        names.Add("James");
        names.Add("Jack");


        // Displaying items
        foreach (string name in names)
        {
            Console.WriteLine(name);
        }
        Console.ReadKey();
    }
}


Expected Output
Alice
Bob
James
Jack


Output Explanation
This program demonstrates creating an ObservableCollection<string> and adding items to it. Each item is then printed, showing the collection holds and displays the added elements.

Conclusion
This lesson covers the basics of creating an ObservableCollection<T> and adding items to it, a fundamental skill for managing dynamic data collections in C# applications.
