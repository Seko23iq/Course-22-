Jagged Arrays in C#
Introduction
Jagged arrays are a special type of array in C# that store arrays as their elements. Unlike multidimensional arrays, each row in a jagged array can have a different length, providing more flexibility.

Declaring a Jagged Array
To declare a jagged array, you use two sets of square brackets. The first set indicates the array, and the second set is for the arrays that will be its elements:

int[][] jaggedArray;

Initializing a Jagged Array
After declaring a jagged array, you must initialize each row:

jaggedArray = new int[3][]; // Jagged array with 3 rows
jaggedArray[0] = new int[4]; // First row has 4 columns
jaggedArray[1] = new int[5]; // Second row has 5 columns
jaggedArray[2] = new int[3]; // Third row has 3 columns


Accessing Elements
You access elements in a jagged array with two indices - the first for the row and the second for the column:

jaggedArray[0][2] = 15; // Sets the third element in the first row to 15


Example Program
Let's create a program that uses a jagged array to store different numbers in each row.

using System;
class JaggedArrayDemo
{
    static void Main()
    {
        // Declare and initialize the jagged array
        int[][] jaggedArray = new int[3][];
        jaggedArray[0] = new int[] { 1, 3, 5, 7, 9 };
        jaggedArray[1] = new int[] { 0, 2, 4 };
        jaggedArray[2] = new int[] { 11, 22 };


        // Display the array elements
        for (int i = 0; i < jaggedArray.Length; i++)
        {
            for (int j = 0; j < jaggedArray[i].Length; j++)
            {
                Console.Write(jaggedArray[i][j] + " ");
            }
            Console.WriteLine();
        }
      Console.ReadKey();
    }
}


Explanation:
Declare and Initialize:
int[][] jaggedArray = new int[3][]; initializes a jagged array with 3 rows.
Assign Values:
Each row is initialized with a different size and values.
Nested Loops for Accessing Elements:
The outer loop (for (int i = 0; i < jaggedArray.Length; i++)) iterates over each row.
The inner loop (for (int j = 0; j < jaggedArray[i].Length; j++)) iterates over each element in the row.
Output:
The array elements are displayed in a matrix form.
Conclusion
Jagged arrays are useful when working with a non-rectangular data structure. They provide more control over the array size of each row, allowing for more efficient memory usage in some scenarios.
