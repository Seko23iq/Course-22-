Checking for Existence in HashSet
Objective
Learn how to check for the existence of an element in a HashSet<T>.

Full Program Example
using System;
using System.Collections.Generic;


class Program
{
    static void Main()
    {
        // Creating and populating a HashSet
        HashSet<string> fruits = new HashSet<string> { "Apple", "Banana", "Cherry" };


        // Checking if "Apple" is in the HashSet
        if (fruits.Contains("Apple"))
        {
            Console.WriteLine("Apple is in the HashSet");
        }
        else
        {
            Console.WriteLine("Apple is not in the HashSet");
        }
    }
}


This code checks whether a specific element is present in the HashSet<T>.
